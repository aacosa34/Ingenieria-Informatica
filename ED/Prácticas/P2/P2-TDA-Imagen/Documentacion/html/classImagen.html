Descripción detallada

T.D.A. Imagen.

Una instancia del tipo de datos abstracto Imagen es una matriz de datos unisigned char compuesta por dos valores enteros que representan, respectivamente, filas y columnas de la matriz. Lo representamos

imagen[filas][columnas]

◆ Imagen() [1/3]

Imagen::Imagen ( )

Constructor por defecto de la clase. Crea la imagen vacía.

Funcion: Imagen(int filas, int cols) Tarea: Crear una imagen en memoria con "filas" filas y "cols" columnas. Reserva memoria para alojar la imagen de "filas" x "cols" pixeles. Recibe: int filas, Número de filas de la imagen. int cols, Número de columnas de la imagen. Devuelve: imagen, la imagen creada. Comentarios:

  1. Operación de tipo constructor.
  2. La imagen creada contiene "filas" filas y "cols" columnas.
  3. La imagen creada no esta inicializada.
